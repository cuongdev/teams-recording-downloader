Connect an AI provider

Configure any OpenAI-compatible endpoint (OpenAI, Azure OpenAI, OpenRouter, local Ollama / LM Studio, …). Used by the in-page "💬 Ask" panel to answer questions about a meeting transcript.

Privacy: When you use the Ask panel with a provider connected, the meeting transcript text is sent to the endpoint you configure above. Only connect an endpoint your organization has approved for meeting content. Your API key is stored locally in this browser profile and is never synced. Without a connected provider the panel runs a local, offline keyword search and sends nothing.
